HTML Template page

1. Select a new image

Put it in the same folder (directory in windows) as this html template. If you call your image image23.png, you will not need to change the name of the image in the template. If you want to change the name of the image do this both for the image itself and in the img src tag in the template. For instance if your new image is called my-image.png in the template change image.png to my-image.png

Either a .jpg or a .png are suggested. To see more about what image types you can use, go here.

2. Substitute your words for the instruction words

Delete the instruction words and in the space where they were, put your own words.

The blue text in these instructions is formatted by the CSS (Cascading Style Sheets) styling listed in the template under the body tag, as follows:
background-color:white;
color:blue;

Try replacing the instructions with your own text. Then, try changing the color in the body tag.

A chart of web safe colors is available here. Note that a CSS a: tag is needed to display the links. (look at the template to see this in action).
